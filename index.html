COMO USAR
📸El sistema funciona por FOTOS. Cada foto es una posicion del line out.
👕Arrastra las camisetas. Clic derecho para orden Z.
🏉Arrastra la pelota. Clic derecho: frente/atras/ocultar por foto.
📝Boton TEXTO en la barra superior para agregar texto al campo. Arrastra la esquina inferior derecha para redimensionar.
↩Ctrl+Z para deshacer.
▶PLAY anima las fotos. Doble clic si queda bloqueado.
🧠Segunda fila de pestanas: CONCEPTOS GENERALES (hilera, maul, etc).
